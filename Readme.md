# Readme

Windows Only

## Description

Block Fighter

Player must kill enemies and duck or block attacks

## Known bugs/problems

sticky controls - attacks hold player in place but delay was done by framecount.

## Keystrokes/Mouse

| Action | Button |
| ------ | ------ |
| Left   | a      |
| right  | d      |
| crouch | s      |
| jump   | w      |
| block  | q      |
| attack | e      |
